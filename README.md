# wava
Core library for Java.\
JDK 1.7 or higher.

## Getting Started

To add a dependency on Wava using Maven, use the following:

```xml
<dependency>
  <groupId>com.wenger.wava</groupId>
  <artifactId>wava</artifactId>
  <version>1.0-SNAPSHOT</version>
</dependency>
```
